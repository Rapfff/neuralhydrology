- Inputs as an element
- edges : 'Element[output] - (input) - Element'
- nodes : 'name : type'
- space characters don't matter
- if one output, '[output]' can be ignored
- if one intput, '(input)' can be ignored
- if several incoming for same input -> sum
- Exactly one out node for each ouput (except for FluxPartition)
- parameters is a iterable of len N. parameter[i] contains an iterable containing the parameters for element i and therefore can be empty.
- can comment by adding a '#' as in python
